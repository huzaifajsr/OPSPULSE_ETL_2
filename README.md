# OpsPulse ETL Pipeline 

## Overview
Containerized ETL pipeline using Python, PostgreSQL, and Docker.

## Tech Stack
- Python (pandas, SQLAlchemy)
- PostgreSQL
- Docker & Docker Compose

## Features
- Extract data from CSV
- Transform (clean + process)
- Load into PostgreSQL
- Fully containerized pipeline

## How to Run
```bash
docker-compose up --build
